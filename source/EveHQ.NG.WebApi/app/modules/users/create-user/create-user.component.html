The application user data

The EveHQ application should know it's user. There is one and only one application user per OS user. User provides authentication possibilities on start which could be skipped if you prefer so.

The user meta-game profiles

Most of the EVE players have no need for a few different meta-game profiles. You need more than one if you, for example, if you manage a corporation as a CEO, lead fleets as fleet commander and play as a single player sometimes. Profile is more about list of characters playing together.

Most of the players need just one.
